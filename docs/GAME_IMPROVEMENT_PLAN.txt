Dpad Arcade - Game Improvement Plan

OVERVIEW:
Standardize user configuration and guidance across all top 15 games based on successful patterns from T-Rex Run and Snake.

GENERIC PATTERNS:
1. Configuration Dialog: Replace manual cycling with a centralized [MENU] dialog.
2. Auto-Hide Quick Hint: Top-left text ("Press [MENU] for Options") for ~5 seconds.
3. State Persistence: Save selected level/mode in SharedPreferences.
4. Draw Loop Optimization: Pre-allocate RectF/Path objects as class members.
5. Per-Selection High Scores: Save and display scores specific to the current mode.
