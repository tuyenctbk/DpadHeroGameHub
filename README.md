# Dpad Arcade: Retro Games 🎮

**Dpad Arcade** is a comprehensive collection of over 20 classic retro games, meticulously optimized for Android devices. Designed with a focus on **D-Pad and Remote Control navigation**, it provides a seamless arcade experience on Android TV, handheld consoles, and devices with physical controllers.

## 🕹️ Included Games

The hub features a diverse library of classic games:

*   **Snake**: The timeless classic - grow your snake without hitting obstacles.
*   **Minesweeper**: Strategic puzzle solving with a visual reveal system.
*   **4096**: A high-stakes variant of the popular 2048 merge puzzle.
*   **T-Rex Run**: Dynamic endless runner with day/night cycles and seasonal weather (Rain/Snow).
*   **Flappy Hero**: Navigate through challenging obstacles with multiple themes (Day, Night, Sunset, Winter).
*   **Brick Break**: Action-packed paddle-and-ball arcade fun.
*   **Tic-Tac-Toe & Checkers**: Essential board games against a smart CPU.
*   **Sokoban & Sudoku**: Challenging logic and spatial puzzles.
*   **Tetris**: The ultimate block-stacking puzzle.
*   **Star Fighter**: Retro space shooter with wave-based combat.
*   **Lines 98**: Strategic color-matching ball game with refined visuals.
*   **Hangman**: Classic word-guessing game.
*   **Road Racer**: High-speed traffic dodging game.
*   **Memory**: Traditional card-matching game to test your recall.
*   **Slide Puzzle**: Classic number/image sliding challenge.
*   **Maze**: Find the correct exit in complex connected paths.
*   **And many more**: Froggy Cross, Battle Tanks, Word Quest, Dungeon Escape, Solitaire, Simon Says, Mental Math, etc.

## ⌨️ Controls & Accessibility

Built for tactile feedback and physical controllers:

| Input | Action |
| :--- | :--- |
| **D-Pad Directionals** | Move cursor, paddle, or character |
| **D-Pad Center / Enter** | Launch ball, select game, or confirm |
| **Back Button** | Return to menu / Exit game |
| **H / Info Button** | Toggle the "How to Play" guide |
| **S / Mute Button** | Toggle in-game sound effects |

## ✨ Key Features

*   **Native Controller Support**: Zero touch-input required. Fully navigable via D-Pad/Remote.
*   **Unified High Scores**: Persistent tracking for all games via `ScoreManager`.
*   **Interactive Guides**: In-game help for every title to ensure players never get stuck.
*   **Immersive Experience**: Automated full-screen mode for distraction-free gaming.
*   **Performance Optimized**: Lightweight custom `Canvas` rendering for smooth 60 FPS gameplay.
*   **Multi-Language Support**: Supports 10+ popular languages including English, Spanish, French, German, Italian, Portuguese, Japanese, Korean, Chinese, Russian, and Vietnamese.
